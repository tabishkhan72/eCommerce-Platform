## Changelog

###

- **Initial Project Setup**: Established the foundational structure of the eCommerce platform using Ruby on Rails.
- **Multi-Functional Support**: Integrated support for multi-vendor, multi-tenant, multi-store, multi-currency, and multi-language functionalities.
- **Core Components**:
  - `admin/`: Administrative interface for managing platform settings and user roles.
  - `api/`: API endpoints facilitating communication between the frontend and backend.
  - `storefront/`: User-facing storefront for browsing and purchasing products.
  - `cli/`: Command-line interface tools for developers and administrators.
  - `docs/`: Documentation detailing setup, usage, and contribution guidelines.
  - `emails/`: Templates and configurations for email notifications and communications.
  - `lib/`: Library modules and utilities supporting various functionalities.
  - `sample/`: Sample data and configurations for testing and demonstration purposes.
- **Configuration and Tooling**:
  - `.circleci/`: Continuous integration setup using CircleCI.
  - `.github/`: GitHub-specific configurations, including issue templates and workflows.
  - `.rubocop.yml`: Ruby code style and linting configurations.
  - `.deepsource.toml`: Static analysis configurations for code quality.
  - `.codeclimate.yml`: Code Climate configurations for maintainability and test coverage.
- **Security and Contribution Guidelines**:
  - `SECURITY.md`: Guidelines for reporting vulnerabilities and security concerns.
  - `README.md`: Overview of the project, setup instructions, and contribution guidelines.
